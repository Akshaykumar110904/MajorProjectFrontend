How to run the project
=> Install the Node.js LTS version
=> Install Git on your machine 
=> Clone the project by typing the command in your git bash
    git clone https://github.com/Akshaykumar110904/MajorProjectFrontend.git
=> Install the packages in your project folder by entering the command in command prompt
    npm i
=> run the project by running the command in command prompt
    npm run dev
Default packages
react, react-dom

Installed packages are 
axios, @reduxjs/toolkit, @tailwindcss/vite, react-redux, tailwindcss, react-router-dom, react-hook-form, react-toastify
